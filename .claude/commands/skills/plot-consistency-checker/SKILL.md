# Plot Consistency Checker Skill

## When to Use This Skill:

User asks to check a story, chapter, or plot for consistency

User wants help finding plot holes, contradictions, or unclear motivations

User is revising fiction and needs structural feedback

User requests evaluation of character arcs, worldbuilding coherence, or timeline logic

## How This Skill Works:

Review the narrative: Read the provided story or excerpt carefully

Identify inconsistencies: Look for plot holes, continuity errors, unclear motivations, pacing issues, or timeline contradictions

Analyze character coherence: Check character behavior, development, and actions for internal logic

Suggest fixes: Offer clear recommendations to resolve inconsistencies and strengthen the story
