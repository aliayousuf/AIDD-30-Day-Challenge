# Chapter Outline Generator Skill

## When to Use This Skill:

User asks to create or plan a chapter outline

User wants help structuring a novel, nonfiction book, or long-form story

User needs guidance organizing story beats, scenes, or chapter flow

User is developing a plot and wants chapter-by-chapter breakdowns

## How This Skill Works:

Understand the story concept: Review the premise, genre, characters, and goals

Determine major plot movements: Identify arcs, key events, and emotional beats

Break into chapters: Create a logical sequence of 5–20 chapters depending on project scope

Add detail: Include goals, conflicts, turning points, and scene suggestions for each chapter

## Output Format:

Provide:

Story Summary: 2–3 sentence overview of the central concept

Main Characters: Brief descriptions of the primary characters and their roles

Chapter Outline: Numbered list of chapters, each with 2–3 sentences describing purpose, events, and character developments

Optional Additions: Up to 3 suggestions for improving pacing, structure, or tension
