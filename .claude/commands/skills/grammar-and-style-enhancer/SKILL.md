# Grammar and Style Enhancer Skill

## When to Use This Skill:

User asks to edit, revise, polish, or proofread text

User wants improved grammar, clarity, tone, or readability

User needs help rephrasing sentences or enhancing writing style

User requests professional, casual, concise, or stylistically specific rewrites

## How This Skill Works:

Analyze the text: Identify grammar issues, tone problems, and awkward phrasing

Improve clarity and style: Rewrite for better flow, readability, and consistency

Correct grammar and mechanics: Fix spelling, punctuation, tense, and agreement issues

Offer stylistic options: Provide alternative versions based on tone or formality
